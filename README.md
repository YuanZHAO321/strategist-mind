# strategist-mind

> 枭雄理性主义谋略分析 —— 一个 Claude Skill
>
> A Claude skill for cold-blooded, rationalist strategic analysis: strip away emotion and moral framing, map the interest structure, and output actionable moves.

用一位虚构枭雄的眼睛看局面:不被情绪蒙蔽,不被道德叙事绑架,只看利益结构与力量对比。适用于职场竞争、谈判博弈、合伙纠纷、重大抉择,也适用于拆解历史事件、小说情节、商业案例中的权谋。

## 它做什么

给它一个局面,它按固定框架推演:

1. **破除迷障** —— 剥掉情绪与道德叙事,还原"到底什么在被争夺"
2. **盘点筹码** —— 资源、信息差、不可暴露的底牌
3. **换位推演** —— 每个相关方:要什么、怕什么、下一步的理性选择;复杂局面可开"多席推演",让谋圣、建设者、识人者、赌手、破壁者五个思维原型各出一手
4. **权衡取舍** —— 逐选项算胜算与代价,含二阶后果
5. **谋定后动** —— 时机、后路、用利益而非人情绑定盟友

最后以**双轨**给出建议:

- **枭雄本色** —— 局中棋手会怎么做(负责看清,可以冷酷)
- **现实转译** —— 你明天就能执行的合法合规版本(负责落地)

两轨落差越大,越说明这个局不值得用激烈手段去打。

除了拆局,也支持**做局**:给它一个目的("我想让 X 发生"),它按布局九步反向设计——先做**目的审计**(我为什么要它?要的是东西还是感觉?得到什么、付出什么、失去什么?只评估、不否决,账本摆清后决定权交还给你),然后定目的、**倒推拆解**(从终点倒推到明天就能做的第一个动作,里程碑各配交付+期限+放弃条件)、盘缺口、给每个关键人选杠杆、用十二条人性情境定律设计机关、排时序、留后路、红队自检。红线内置:阳谋经得起公开,这既是道德线也是工程质量线。

## 示例

```
> 同级同事把我做的方案包装成他的想法拿去汇报了,我不想撕破脸也不想继续吃亏,帮我分析。

> 前同事拉我去他的创业公司当联创,股份 8%、现金砍半,该去吗?

> 分析刘备"借荆州不还"这步棋,是妙手还是败笔?
```

## 安装

**Claude Code(个人级)**:

```bash
git clone https://github.com/<you>/strategist-mind.git ~/.claude/skills/strategist-mind
```

**Claude Code(项目级)**:放到项目的 `.claude/skills/strategist-mind/`。

**零安装(即插即用)**:把仓库里的 `CLAUDE.md` 复制到任意项目根目录,该项目内的决策类问题即默认采用本 Skill 的精简思维流程,无需安装技能。

**其他 LLM / Agent 环境**:`AGENT.md` 是厂商无关的执行协议——直接粘贴给任意 agent 作系统指令,或放进遵循 `AGENT.md`/`AGENTS.md` 约定的工作目录,脱离 Claude Code 也能复现完整流程(拆局六步 + 做局九步 + 双轨输出)。

安装后无需配置。当你要求"冷酷理性分析""用枭雄/反派视角看"或描述一个博弈局面时,skill 自动触发;也可以直接点名使用。

## 谋略十二式一览

| # | 模式 | 一句话 |
|---|---|---|
| 一 | 底牌管理 | 底牌的价值在于对方不知道 |
| 二 | 利益枷锁 | 合作的稳固度 = 对方背叛你的机会成本 |
| 三 | 换位推演 | 用对方的动机做杠杆,而非对抗其行为 |
| 四 | 时机忍耐 | 预埋条款,等触发条件,后发制人 |
| 五 | 果断止损 | 沉没成本永不参与决策 |
| 六 | 风险计算 | 先算被反噬的概率,再看收益 |
| 七 | 孤注一掷 | 只在退无可退或赔率极端悬殊时 |
| 八 | 阳谋 | 让对方明知是局也要走进来 |
| 九 | 信息差 | 独有认知是资产,且有保质期 |
| 十 | 绝境反打 | 生路藏在对手的预判之外 |
| 十一 | 变局取势 | 每一次洗牌都是弱者的机会 |
| 十二 | 借势 | 让既有的力量替你做功 |

## 理论对照

库中的经验法则大多能挂到成熟理论上,可放心迁移:凡事有价 ≈ 机会成本/期望值;变局取势与灾祸磨砺 ≈ 反脆弱(Taleb);创新即重组 ≈ 熊彼特;借势让人自愿 ≈ 机制设计;意义可被赋予 ≈ 存在主义;认知三层 ≈ 第一性原理;复盘回校 ≈ 贝叶斯更新 / After-Action Review;决策分级 ≈ 可逆性"单向门/双向门"。
