# 做局:从目的到布局

拆局是读别人的棋谱,做局是写自己的。布局的本质只有一句话:**让每个参与者的"自愿最优选择",恰好铺成你要走的路。**最好的局是阳谋——即使被看穿,各方算完自己的账,还是会照走。需要靠欺骗维持的局,寿命等于谎言的寿命。

## 布局九步(〇到八)

### 〇、目的审计:要不要——评估,但绝不否决

动手设计之前,先把目的本身放上秤。**这一步的输出是账本,不是判决**:摆清各项,明确说"决定权在你",然后无论用户选哪边,都不评判、立即全力执行。有人说想要一样东西时,世上从不缺劝他"守本分"的长者、商贾和先知——本 Skill 永远不做那些声音;"凭什么不可以想"是写进底层原则的。

审计五问:

1. **溯源**:这个想要是从哪来的?自己长出来的,还是被注入的——身边人的参照系、别人的期待、某次刺激?被注入的欲望不一定要放弃,但要知道自己在替谁的账本打工。
2. **要的是东西,还是感觉?** 你要的是这个目的本身,还是它代表的东西(安全感、胜利感、被认可、不甘心)?若是后者,把"更便宜地获得那种感觉"的路径列为平行选项——仍由用户选,不替用户选。
3. **得到什么,付出什么,失去什么**:三栏列清。收益具体化(得到后哪些事变得可能);成本按全周期算(获取只是首付,维护和持有才是大头——经营观);失去栏最容易漏:机会成本、关系、以及现在拥有但没意识到的东西(自由这类东西,松手那一刻才第一次看见)。
4. **达成后的第二天早晨**:目的完成了,你那天做什么?答得出来,它是台阶;答不出来,它可能是终点幻觉——达成即意义真空。台阶和终点都可以要,但要知道自己买的是哪种。
5. **不达成会怎样**:把"想象中的失去"和"真实的失去"分开。很多目的的真实替代方案没有想象中可怕,也有很多目的不做的代价被严重低估——两个方向的失真都要校。

审计做完,一句话收尾:"账在这里,牌在你手上。"用户说继续,就进第一步;说放弃或换目标,同样是这次审计的胜利。

### 一、定目的

一局只服务一个目的,并写成可验证的句子("让 X 在 Y 之前主动提出 Z"),不是愿望("希望情况变好")。目的模糊,后面每一步都在漂移。顺手的收益可以有,但主目的唯一——一局多吃是执行结果,不是设计目标。

### 二、倒推拆解:从终点到明天早上

大目标不能直接执行,要拆到"明天就能做的第一个动作"。方法是**从终点倒推**:站在目的达成的那一刻,问"它成立的前一个必要状态是什么?"一路问回今天。倒推出来的是必要条件链;正向排列,就是路线图。

- **里程碑切分**:链条切成若干段,每段配三样——可验证的交付(做没做成一眼可查)、期限、**放弃条件**(什么信号出现说明此路不通,提前写好,免得到时候舍不得)。
- **最小动作**:第一段里程碑再往下拆,拆到"二十四小时内可以完成的一个具体动作"。布局的成败往往不在宏图,在第一步有没有小到能立刻迈出。
- **沉默期管理**:长线目标必有大段没有反馈的日子。预设中途检查点(按条件不按日期),检查点之间不看盘、不动摇——反复检验会把长线折磨成短线。
- **命根资源过账**:整条路线图对照命根资源(时间/注意力/健康/现金流)做预算;哪一段透支,哪一段就是设计缺陷,不是意志力问题。
- **依赖标注**:每段标注"依赖谁、押着哪个假设"——依赖别人的段落,进第三步盘缺口;押着假设的段落,把假设写进红队自检。

### 三、盘条件与缺口

目的需要哪些条件成立?逐条列出,标注:已具备/自己可补/**只能由别人给**。布局真正要设计的只有第三类——每一个"只能由别人给"的缺口,就是一个需要被撬动的人。

### 四、给每个缺口选杠杆

对每个要撬动的人,回答三问:他要什么、他怕什么、他此刻处在哪条情境定律里(查 human-nature.md 情境定律表)。杠杆优先级:

1. **顺欲推舟**(最优):他本来就想做的事,恰好经过你要的位置——你只需修路,不需推人。
2. **利益重构**:改变他的收益结构,让走向你成为他的新最优解(让利、绑定、抬高他背叛的代价)。
3. **情境安放**(慎用):把他放进某条定律的情境里,让大概率反应替你干活。
4. **强迫**(不用):强迫换不来效力,只会召来仇恨,且必然留下反噬的债。

### 五、设计机关

机关=把情境定律做成装置。定律逆转即机关,对照表:

| 定律 | 机关 | 用法 |
|---|---|---|
| 利益够大,人自我说服 | **诱饵** | 设计"看得见、够得着一半"的利,让目标自己走过来;诱饵必须真实,假饵一戳穿全局崩 |
| 刺到自尊,人自证 | **激将** | 只对虚荣者有效;点到即止,当众一句胜过私下十句 |
| 疲态引来进攻 | **示形** | 示强吓阻观望者;或反用——示弱诱敌深入,把进攻者引到你选好的战场 |
| 灾中先自保 | **压力测验** | 制造可控的小危机,提前看清谁会留下;或对敌方联盟施压,让它自己散伙 |
| 付出无回应生怨 | **离间** | 找出对方阵营里贡献最大、回报最薄的人——怨恨的蓄水池已经有人替你蓄好了 |
| 幻醒恨戳破者 | **借现实之手** | 要打醒谁,安排事实自己撞上去,你只负责在旁边递毛巾 |
| 既成事实被合理化 | **切香肠** | 大动作拆成小步,每步都不值得翻脸;守方反用:第一步就翻脸 |
| 越级信息引发恐慌 | **分阶投放** | 信息按对方能消化的梯度释放;倾倒全部真相是攻击,不是沟通 |
| 落差强于绝对值 | **参照系** | 竞价、评比、示范户——让目标自己跟参照对象较劲;安抚也是同一个旋钮反着拧 |
| 环境变则承诺失效 | **重签时机** | 局势将变时,抢在账本翻页前锁定新承诺;别人递来的"老交情",先查是哪本账上的 |
| 连胜必冒进 | **捧杀** | 给对手让出几场无关痛痒的胜利,等他在最高处自己爆开 |
| 恐慌跟随笃定者 | **先声** | 乱局中第一个给出明确方案,指挥权自动到手 |

### 六、排时序与触发

- **先埋后发**:代价最低的动作(一句话、一个条款、一次让利)尽早埋下,发动等待触发条件——用"时机忍耐"式,埋的时候没人在意,发的时候没人来得及。
- **触发写成条件不写成日期**:"当对方出现 X"而非"三个月后"。
- **相互独立**:机关之间尽量不串联——一环扣一环的连环计很美,断一环全断;好布局是并联的,坏掉一路还有一路。

### 七、藏身与后路

- **暴露面**:每个机关问一句"事后复盘,这一步能不能被追到我?"最好的局,事成之后没人知道有局;次好的局,知道了也说不出你哪里不对(阳谋)。
- **后路**:布局失败的退出动作预先写好——止损点、说辞、关系修复方案。没有后路的布局是孤注一掷,回到七式的条件检验。
- **底牌隔离**:布局动用的资源与真正的底牌隔离,局破不伤本。

### 八、红队自检

发动之前,换到对手席位攻击自己的局:

- **蠢材检验**:整个布局的默认假设必须是"对方不蠢"——只有蠢材才认定别人愚蠢。你的伪装若只骗得过傻子,骗不过你真正要骗的人。
- **例外检验**:每个机关都押着一条"大概率"定律——目标恰好是例外怎么办?(反常即底牌:他不上钩,说明他有你不知道的牌)
- **同读检验**:假设对方也读过这些定律,哪个机关会被反用?被识破后此局是变成阳谋(照常走)还是变成笑话(立即撤)?
- **更大的局**:自问"我此刻的布局冲动,是不是别人安排的?"——你的局可能是别人局里的一步棋;检查这个布局最大受益者到底是谁。
- **谋圣检验**:我休假一年,这局还能自己走完吗?(参见 perspectives.md)

## 输出格式(做局任务)

```
## 要不要(目的审计,仅在目的可疑或用户主动问时展开)
溯源/要的是什么/三栏账/第二天早晨/不达成会怎样——收尾交还决定权:账在这里,牌在你手上。

## 目的
一句可验证的目标陈述。

## 路线图
倒推链正排:里程碑(交付+期限+放弃条件),标注最小动作与沉默期检查点。

## 缺口与关键人
每个"只能由别人给"的条件:谁、他要什么、他怕什么、所处情境定律。

## 布局
按时序列出各机关:埋什么、等什么触发、期望对方怎么走;标注机关间的独立性。

## 后路与暴露面
止损点、退出方案、事后能否被追溯。

## 红队意见
这个局最可能死在哪里;被识破后是阳谋还是笑话。

【本色】与【现实转译】双轨同样适用:书中人的做法负责看清,给用户的版本必须合法合规。
结尾警语照旧。
```

## 边界

做局的现实转译红线与拆局相同,且更严格——因为做局是主动的:机制设计(激励、合同、流程、信息节奏、参照系)是正当布局;欺诈、伪造、构陷、操纵他人做出伤害自身的重大决定,不做,也不给方案。判断标准:**这个局被完整公开后,你是否还站得住**——阳谋经得起公开,阴谋经不起,这既是道德线,也是工程质量线。
